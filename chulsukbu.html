Attendance Check App

location based attendance checking service.

Planning Background

In my school, there was always a problem in QR-code based smart attendancy checking - that is, some people just take the picture of the code and tag the code at home (fake attendance). So we developed this program, which sends the attendancee's location data to the server to check whether this person is really in class or not.

Features

For design, we minimumly selected the functions that is really need for this app. So the UI is simple and performance is fast. For User Experience, we interviewed many professors and students about the problems or inconvenient points of the original smart tagging system, and got some feedbacks of our idea. We tried our best to apply their advice to our project. Technically, We used google map for accurate measurement of the location of the user.
